## 1. 타겟 고객 페르소나

### 공급측 (은퇴 전문가) - 3개 핵심 페르소나

#### 페르소나 A: "김 사부" - 제조업 숙련 기술자
- **프로필:** 55-70세, 금형/용접/CNC 등 제조 현장 30년+ 경력, 최근 퇴직
- **동기:** "내가 쌓은 기술이 사라지는 게 아깝다", 은퇴 후 소일거리와 부수입 원함
- **고통점:** 유튜브/블로그는 어렵고, 컨설팅은 체력적으로 부담, 지식 전달 방법이 없음
- **디지털 역량:** 스마트폰 카카오톡 수준. PC 사용 제한적
- **핵심 니즈:** "말로 하면 되는" 간단한 방식으로 내 노하우를 남기고 싶다
- **설득 포인트:** "인터뷰에 답하기만 하면 됩니다. 나머지는 저희가 합니다."

#### 페르소나 B: "박 교수" - 은퇴 전문직/학자
- **프로필:** 60-75세, 의료/법률/공학 분야 교수 또는 전문가, 퇴임 후 활동 감소
- **동기:** 학문적 유산을 남기고 싶음, 후학 양성에 대한 사명감
- **고통점:** 책은 이미 썼지만 "실무 판단"은 책에 담기지 않음
- **디지털 역량:** 중상. 이메일, 줌 사용 가능
- **핵심 니즈:** 내 판단 체계와 경험적 지혜를 구조화해서 전수하고 싶다
- **설득 포인트:** "교과서에 없는 당신의 판단 노하우가 후배들에게 전달됩니다."

#### 페르소나 C: "최 셰프" - 전통 기술/장인
- **프로필:** 50-70세, 한식 요리/전통 공예/농업 등 장인, 도제식 전수 한계 인식
- **동기:** 전통 기술 단절에 대한 위기감, 제자가 줄어드는 현실
- **고통점:** 가르칠 제자가 없음, 유튜브는 "감"을 전달하기 어려움
- **디지털 역량:** 낮음~중간
- **핵심 니즈:** 내 기술의 "감"과 "판단"을 다음 세대에 넘기고 싶다
- **설득 포인트:** "50년 경력의 감각, AI가 기억합니다."

### 수요측 (지식 소비자) - 3개 핵심 페르소나

#### 페르소나 D: "이 대리" - 제조업 중소기업 실무자
- **프로필:** 28-40세, 중소기업 생산/기술팀, 사수 없이 혼자 해결해야 하는 상황
- **동기:** 현장 판단이 필요한데 물어볼 사람이 없음 (숙련공 퇴직)
- **고통점:** 매뉴얼에는 없는 "이럴 땐 이렇게" 판단이 필요, 외부 컨설팅은 비쌈
- **지불 의사:** 월 5-20만원 (회사 비용처리 가능)
- **핵심 니즈:** "이 상황에서 경험 많은 선배라면 어떻게 판단할까?"
- **설득 포인트:** "30년 경력 선배의 판단을, 24시간 옆에 두세요."

#### 페르소나 E: "정 팀장" - 기업 교육/인사담당자
- **프로필:** 35-50세, 중견/대기업 HRD 또는 기술교육 담당
- **동기:** 숙련 인력 퇴직에 따른 기술 전수 시스템 구축 필요
- **고통점:** 은퇴자 재고용은 비용/법적 제약, OJT는 비체계적
- **지불 의사:** 페르소나당 500-2,000만원 (기업 교육예산)
- **핵심 니즈:** 퇴직 전문가의 암묵지를 조직 자산으로 보존하고 싶다
- **설득 포인트:** "핵심 인력의 30년 노하우, 퇴직과 함께 사라지지 않습니다."

#### 페르소나 F: "한 원장" - 전문 서비스 소비자
- **프로필:** 30-55세, 중소 병원장/법무사/농업인 등 전문 조언이 필요한 개인
- **동기:** 특정 상황에서 경험 많은 전문가의 판단이 필요하지만 접근이 어려움
- **고통점:** 대학병원 교수/석학에게 직접 자문받기 어려움, 비용 부담
- **지불 의사:** 건당 또는 월 10-50만원
- **핵심 니즈:** "전문가의 관점에서 이 상황을 어떻게 봐야 하나?"
- **설득 포인트:** "은퇴한 명의의 40년 임상 판단을 참고하세요."

## 2. 포지셔닝

### "AI 챗봇"이 아닌 것으로 포지셔닝해야 하는 이유

"AI 챗봇"은 고객 서비스, 단순 Q&A의 이미지. Human Archive AI의 본질은 **"전문가의 판단 체계 이식"**이다.

### 포지셔닝 프레임워크

```
카테고리: 전문가 판단 아카이빙 플랫폼
경쟁 대안: 컨설팅, 재고용, OJT, 매뉴얼
차별화: "경험"과 "판단"을 구조화하여 AI에 이식
```

### 포지셔닝 스테이트먼트

> **Human Archive AI**는 **은퇴 전문가의 판단 체계와 의사결정 노하우**를 AI에 이식하는 **전문가 지혜 아카이빙 플랫폼**입니다.
>
> 단순 정보 검색이 아니라, 특정 전문가가 **"이 상황에서 어떻게 판단할 것인가"**를 재현합니다.

### 포지셔닝 맵

| | 판단 체계 보존 (높음) | 판단 체계 보존 (낮음) |
|---|---|---|
| **접근성 높음** | **Human Archive AI** | 일반 AI 챗봇 (ChatGPT) |
| **접근성 낮음** | 컨설팅/재고용 | 매뉴얼/교재 |

### 핵심 차별화 메시지

- **vs. ChatGPT/일반 AI:** "ChatGPT는 평균적 답을 줍니다. 우리는 '그 사람'의 답을 줍니다."
- **vs. 컨설팅:** "시간당 50만원 컨설팅을 월 구독으로."
- **vs. 매뉴얼/교재:** "매뉴얼에 없는 '이럴 땐 이렇게'를 답합니다."
- **vs. 유튜브 강의:** "영상은 일방향입니다. 우리는 당신의 상황에 맞게 답합니다."

## 3. GTM (Go-To-Market) 전략

### Phase 1: 파일럿 (0-6개월) - "증거 만들기"

**목표:** 3-5개 성공 사례 확보, PMF(Product-Market Fit) 검증

**전략: 제조업 숙련 기술 이전 특화**

선택 이유:
- 한국 제조업의 숙련공 퇴직 문제는 이미 사회적 이슈 (뉴스, 정부 정책)
- 문제의 시급성이 높아 지불 의사가 큼
- "금형 설계 30년 경력" 같은 구체적 사례가 스토리텔링에 유리
- 중소기업 정부 지원금/바우처 활용 가능

**실행 계획:**

| 주차 | 액션 | 목표 |
|---|---|---|
| 1-4 | 제조업 협회/폴리텍 접촉, 파일럿 전문가 3명 섭외 | 무료 제작 제안 |
| 5-8 | 인터뷰 기반 페르소나 제작 (금형, 용접, CNC 각 1명) | 페르소나 완성 |
| 9-12 | 해당 기업 실무자에게 무료 사용 제공 | 사용 데이터 수집 |
| 13-20 | 사례 영상/보고서 제작, 효과 측정 | Before/After 증거 |
| 21-24 | 유료 전환, 추가 기업 영업 | 유료 고객 5개사 |

**파일럿 성공 지표:**
- 실무자 주 3회 이상 사용
- "이 답변이 도움이 됐다" 긍정 피드백 70%+
- 기업 담당자가 유료 전환 의사 표시

### Phase 2: 확장 (6-18개월) - "시장 입증"

**수직 확장 (제조업 내):**
- 금형 → 용접 → CNC → 품질관리 → 설비보전 순으로 확대
- 업종별 대표 전문가 페르소나 10-20개 확보
- 중소기업 기술 바우처 사업 공식 등록

**수평 확장 (인접 시장):**
- 전통 식문화 (명인/장인의 레시피와 판단)
- 한의학/대체의학 (임상 경험 기반 판단)
- 농업 (경험 기반 재배 판단)

### Phase 3: 플랫폼화 (18개월+) - "마켓플레이스"

- 전문가가 직접 등록하는 셀프서비스 도구 제공
- 수요-공급 매칭 마켓플레이스
- MCP 서버 기반 타 AI 플랫폼 연동 (Claude, ChatGPT 등에서 페르소나 호출)
- 기업용 사내 전용 배포 (On-premise/Private Cloud)

## 4. 고객 획득 채널

### 공급측 (은퇴 전문가) 획득 채널

| 채널 | 구체적 방법 | 예상 효과 | 비용 |
|---|---|---|---|
| **폴리텍/기술교육원** | 은퇴 교수/강사 대상 설명회. "당신의 강의가 영원히 남습니다" | 높음 | 낮음 |
| **제조업 협회** | 한국금형공업협동조합, 대한용접학회 등 MOU | 높음 | 중간 |
| **시니어 일자리 센터** | 고용노동부 시니어 인턴십 프로그램과 연계 | 중간 | 낮음 |
| **은퇴자 커뮤니티** | 시니어 카페(다음/네이버), 실버넷, 대한은퇴자협회 | 중간 | 낮음 |
| **직접 접촉** | 명장/명인 수상자 리스트 기반 직접 제안 | 높음 (소수) | 중간 |
| **기존 고객 소개** | 아카이빙 완료 전문가 → "같은 업계 아시는 분 소개" | 높음 | 매우 낮음 |
| **언론/PR** | "사라져가는 기술을 AI로 보존" 사회적 가치 스토리 | 인지도 | 중간 |

**핵심 전략: "첫 전문가는 직접 찾아가라"**
- 초기 5명은 무료로 제작, 최고의 퀄리티로 사례 확보
- 전문가 본인이 만족하면 최고의 영업사원이 됨
- "우리 업계에서 이런 거 했대" → 동종업계 자연 확산

### 수요측 (지식 소비자) 획득 채널

| 채널 | 구체적 방법 | 예상 효과 | 비용 |
|---|---|---|---|
| **기업 직접 영업** | 중소기업 기술교육 담당자 콜드 이메일/방문 | 높음 | 중간 |
| **정부 사업 연계** | 스마트공장 사업, 기술 바우처, R&D 바우처 | 매우 높음 | 시간 투자 |
| **산업 전시회** | SIMTOS(공작기계), KOMAF(제조) 등 부스/세미나 | 중간 | 높음 |
| **LinkedIn/기술 블로그** | 제조업 기술 전수 사례 콘텐츠 → 리드 수집 | 중간 | 낮음 |
| **SEO/검색** | "숙련공 퇴직 대비", "기술 전수 방법" 키워드 | 장기적 높음 | 낮음 |
| **파트너십** | 제조 MES/ERP 업체와 번들, 기술교육 업체와 제휴 | 높음 | 중간 |

**핵심 전략: "문제를 가장 심하게 느끼는 곳부터"**
- 최근 숙련공이 퇴직한 중소기업 (뉴스/구인공고로 파악 가능)
- 기술 전수 관련 정부 사업 수혜 기업
- 폴리텍/직업훈련원 재직자 교육 프로그램

## 5. 가격 전략

### 수익 모델 구조

```
[1] 페르소나 제작비 (일회성) + [2] 구독료 (월정액) + [3] 전문가 수익 분배
```

### [1] 페르소나 제작비

| 등급 | 인터뷰 깊이 | 가격 | 대상 |
|---|---|---|---|
| **라이트** | AI 인터뷰 9개 질문 (현재 기본) | 무료-50만원 | 개인 전문가 체험 |
| **스탠다드** | 인터뷰 + 유튜브/강의 아카이빙 5-10건 | 200-500만원 | 개인/소규모 |
| **프리미엄** | 심층 인터뷰(대면 3회) + 영상 20건+ + 전문 편집 | 1,000-3,000만원 | 기업/기관 의뢰 |
| **엔터프라이즈** | 다수 전문가(5명+) 패키지 + 사내 배포 | 협의 (5,000만원+) | 대기업/공공기관 |

**초기 전략:** 라이트 무료 + 스탠다드 파일럿 할인(100만원)으로 사례 확보

### [2] 구독료 (페르소나 사용)

| 플랜 | 가격/월 | 내용 | 대상 |
|---|---|---|---|
| **개인** | 9,900원 | 페르소나 1개, 월 100회 대화 | 개인 학습자 |
| **프로** | 49,000원 | 페르소나 3개, 무제한 대화, API 키 | 프리랜서/소기업 |
| **팀** | 199,000원 | 페르소나 10개, 5인 시트, 사용 분석 | 중소기업 팀 |
| **엔터프라이즈** | 협의 (월 100만원+) | 무제한, 사내 배포, SLA, 전용 모델 | 대기업 |

### [3] 전문가 수익 분배

| 항목 | 전문가 몫 | Human Archive 몫 |
|---|---|---|
| 구독 수익 | **40%** | 60% |
| 최소 보장 | 월 10만원 (유료 구독자 발생 시) | - |
| 제작비 수익 (기업 의뢰) | **30%** | 70% (제작 비용 포함) |

**설계 근거:**
- 전문가에게 40% 분배는 "내 지식이 돈이 된다"는 동기 부여 핵심
- 월 10만원 최소 보장은 심리적 안전망 (구독자 10명 이상 시 자동 초과)
- 기업 의뢰 30%는 전문가 참여 인센티브 (3,000만원 제작 시 900만원)

### 가격 앵커링 전략

"30년 경력 전문가 컨설팅 시간당 50만원 × 월 4회 = 200만원/월" vs. "AI 페르소나 구독 월 49,000원"
→ **97.5% 비용 절감** 메시지

- 공식 브랜드: Human Archive (기업/글로벌용)

## 7. 콘텐츠 마케팅

### 콘텐츠 전략 프레임워크

```
인지 → 관심 → 신뢰 → 전환
```

### Phase 1: 인지도 확보 (0-6개월)

#### (1) 파운더 스토리 / 미션 콘텐츠
- **"왜 우리는 은퇴 전문가의 지혜를 AI에 담으려 하는가"** - 창업 동기 에세이
- 브런치/미디엄에 발행, LinkedIn 공유
- 목표: 공감 확보, 초기 팔로워 500명

#### (2) 문제 제기 콘텐츠 시리즈
- **"사라지는 기술들"** 시리즈 (월 2회)
  - "한국 금형 산업, 숙련공 퇴직 후 무슨 일이 일어나는가"
  - "30년 경력 용접사가 떠난 후, 불량률 300% 증가한 사연"
  - "전통 장인의 마지막 제자가 포기한 이유"
- 네이버 블로그 + 유튜브 숏폼 (인터뷰 클립)

#### (3) 데모 콘텐츠
- **"AI에게 금형 설계를 물어봤다"** - ChatGPT vs. Human Archive 비교
- 실제 현장 질문으로 비교 → "일반 AI는 교과서, Human Archive는 현장"
- 유튜브 + LinkedIn 발행

### Phase 2: 신뢰 구축 (6-12개월)

#### (4) 전문가 인터뷰 시리즈
- **"명장의 마지막 수업"** - 아카이빙 과정 다큐멘터리
- 전문가가 인터뷰에 답하는 모습 + AI 페르소나 대화 장면
- 유튜브 롱폼 (15-20분), 숏폼 클립 제작

#### (5) 고객 사례 (Case Study)
- **"금형 설계 신입이 AI 사부와 함께한 6개월"**
- Before/After 데이터 포함 (불량률, 판단 속도, 교육 비용)
- PDF 다운로드 + 웨비나

#### (6) 기술 콘텐츠
- **"전문가의 암묵지를 어떻게 AI에 이식하는가"** - 기술 블로그
- 인터뷰 → 원칙 추출 → RAG → 판단 재현 프로세스 공개
- 개발자/기술 의사결정자 대상

### Phase 3: 전환 최적화 (12개월+)

#### (7) ROI 계산기
- "숙련공 1명 퇴직 시 연간 손실 비용 vs. Human Archive 비용" 웹 도구
- 입력: 업종, 인원수, 숙련공 비율 → 출력: 예상 절감액

#### (8) 무료 체험 콘텐츠
- "우리 회사에 맞는 전문가 페르소나 1개 무료 제작" 이벤트
- 리드 수집 → 영업 파이프라인 연결

### 채널별 콘텐츠 배분

| 채널 | 콘텐츠 유형 | 빈도 | 목적 |
|---|---|---|---|
| **유튜브** | 전문가 인터뷰, 데모, 비교 영상 | 주 1회 | 인지도/신뢰 |
| **네이버 블로그** | SEO 콘텐츠, 사례, 업계 분석 | 주 2회 | 검색 유입 |
| **LinkedIn** | 파운더 인사이트, 기업 사례, 기술 글 | 주 3회 | B2B 리드 |
| **브런치** | 스토리텔링, 전문가 이야기 | 월 2회 | 브랜드 감성 |
| **뉴스레터** | 업계 동향 + 제품 업데이트 | 월 2회 | 리텐션/전환 |

## 8. 바이럴 요소

### 내재적 바이럴 루프

#### (1) "내 사부" 공유 루프
- 수요자가 페르소나와 대화 후 유용한 답변을 받으면 → **"공유하기"** 버튼
- 공유 시 메시지: "30년 경력 금형 전문가에게 물어봤더니 이런 답을 줬어요"
- 공유 카드에 전문가 프로필 + 답변 하이라이트 포함
- **K-factor 목표: 0.3** (사용자 3명 중 1명이 1명에게 공유)

#### (2) "나도 아카이빙" 전문가 확산 루프
- 아카이빙 완료된 전문가에게 → "같은 분야 동료에게 추천" 기능
- **인센티브:** 추천인이 아카이빙하면 추천자 구독 수익 +5%
- 같은 업종/학회/협회 내 자연 확산

#### (3) "전문가 랭킹" 경쟁 루프
- 페르소나별 사용 횟수, 만족도 공개 랭킹
- "이달의 인기 사부" 선정 → 전문가의 자부심 + 홍보 효과
- 전문가 본인이 SNS에 공유: "제가 1위에요!"

### 외부 바이럴 트리거

#### (4) 감성 스토리텔링
- **"돌아가신 아버지의 AI"** 같은 스토리 (단, 서비스 본질은 "전문 지식 보존")
- "은퇴한 스승님의 AI와 대화했습니다" 체험기 → SNS 자연 확산
- 감정적 공감 → 공유 → 서비스 인지

#### (5) 업계 챌린지
- **"#내스승님챌린지"** - "은퇴한 스승님/선배님을 추천해주세요"
- 추천받은 전문가에게 무료 아카이빙 제공 (월 3명 한정)
- 추천자와 피추천자 모두에게 이야기가 생김 → 자연스러운 입소문

#### (6) 언론/PR 바이럴
- **"사라지는 한국의 장인 기술, AI가 지킨다"** - 사회적 가치 프레이밍
- KBS/MBC 다큐 제안, 조선일보/한겨레 기획 기사 제안
- 정부 정책(기술 전수, 시니어 일자리)과 연계한 보도자료

#### (7) 개발자 바이럴 (MCP 생태계)
- MCP 서버 오픈소스 공개 → 개발자 커뮤니티 확산
- "Claude에서 금형 전문가 AI를 불러서 썼다" 개발자 체험기
- GitHub 스타 + 테크 블로그 소개 → 기술 커뮤니티 인지도
