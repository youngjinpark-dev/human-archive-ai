## 3. 한국 시장 특수성

### 3.1 고령화 - 세계 최고 속도

- 2025년 65세 이상 인구 **20.3%** 돌파 → **초고령사회 공식 진입**
- 2026년 고령인구 **1,084만명 (21.21%)**
- 2036년 **30%**, 2050년 **40%** 전망
- 고령화사회 → 초고령사회 전환: **단 7년 4개월** (세계 최단)
- 국민연금 수급자 2026년 **800만명** 돌파

### 3.2 베이비붐 세대 대량 은퇴 = 지식 유출의 위기

- 1차 베이비부머 (1955~1963년생, 705만명): **은퇴 완료**
- 2차 베이비부머 (1964~1974년생, 954만명): **2025년부터 11년간 순차 은퇴**
- 매년 약 **80~90만명**의 숙련 인력이 산업 현장에서 이탈
- 이들의 **암묵지(tacit knowledge)** 는 매뉴얼로 전수 불가

### 3.3 기술인력 부족과 기술 단절

- 제조업 근로자 평균 연령 **40대 초과**
- 뿌리산업(주조/금형/용접/소성가공/표면처리/열처리) 6대 분야 기술 단절 심각
- 청년층 기술직 기피 → 숙련공 세대교체 실패
- 전문가 경고: **"지금 대응하지 않으면 숙련 공백과 기술 단절이 고착화"**

### 3.4 도제식 전수 문화의 디지털 전환 기회

한국의 전통적 기술 전수 방식:
- **도제 관계**: 1:1 사수-부사수, 수년간 현장 체험
- **암묵지 중심**: "감으로 안다", "손끝에 있다" — 언어화 어려운 경험적 판단
- **세대 간 괴리**: MZ세대는 도제식 학습 거부, 기존 세대는 디지털 적응 어려움

**Human Archive AI의 기회**: 도제식 전수의 **디지털 브릿지** 역할
- 은퇴 전문가의 판단 체계를 구조화된 AI로 변환
- "30년 경험의 금형 장인에게 물어보기" → MCP 서버로 24/7 접근 가능

## 5. 진입 장벽과 해자(Moat) 분석

### 5.1 Human Archive AI가 구축 가능한 해자

| 해자 유형 | 강도 | 설명 |
|-----------|------|------|
| **데이터 해자** | **강** | 전문가별 고유 판단 체계 데이터는 복제 불가. 한번 아카이빙된 전문가 데이터는 경쟁사가 재수집 불가능 |
| **네트워크 효과** | 중 | 전문가 수 증가 → 이용자 증가 → 더 많은 전문가 유입의 양면 네트워크 |
| **전환 비용** | **강** | 기업이 특정 전문가 AI에 의존하면 전환 비용 높음 (워크플로우 통합, MCP 연동) |
| **브랜드/신뢰** | 중 | "검증된 전문가" 브랜딩, 전문가 선별 과정의 신뢰 |
| **기술적 해자** | 약~중 | RAG/임베딩 기술 자체는 범용적이나, 인터뷰→구조화→MCP 파이프라인은 노하우 |

### 5.2 잠재적 위협

| 위협 | 심각도 | 대응 |
|------|--------|------|
| 대형 AI 기업(OpenAI, Google)의 유사 기능 출시 | 높음 | 한국 시장 특화 + 전문가 네트워크 선점이 핵심 방어 |
| 클래스101 등 기존 플랫폼의 AI 전환 | 중간 | 이미 보유한 크리에이터 풀 위협적이나, "판단 체계 이식"과 "강의 녹화"는 본질적으로 다름 |
| LLM 자체 능력 향상으로 전문가 AI 필요성 감소 | 중간 | 범용 LLM은 "30년 금형 경험" 같은 현장 암묵지를 학습할 수 없음 |

### 5.3 해자 강도 종합 평가: **중상(Medium-High)**

가장 강력한 해자는 **전문가 데이터의 비복제성**. 한번 독점 계약으로 아카이빙한 "김철수 마이스터의 30년 금형 판단 체계"는 경쟁사가 재현할 수 없다. 이것이 Character.AI나 범용 LLM과의 본질적 차이.

## 6. 시장 진입 타이밍 분석

### 6.1 타이밍 매트릭스

| 요인 | 상태 | 판단 |
|------|------|------|
| 기술 성숙도 | RAG, 임베딩, LLM 모두 상용 수준 도달 | **적절** |
| MCP 생태계 | 2024년 발표 → 2026년 10,000+ 서버, 업계 표준화 | **적절 (골든타임)** |
| 수요 신호 | 2차 베이비부머 은퇴 시작 (2025~), 기술 단절 경고 | **적절** |
| 경쟁 상황 | 직접 경쟁사 한국 내 부재, 글로벌도 피봇 중 | **적절 (선점 기회)** |
| 사회적 인식 | AI 활용 보편화, "디지털 레거시" 개념 확산 | **적절** |
| 규제 환경 | 개인정보보호 강화 추세이나 직접 규제 부재 | **주의 필요** |

### 6.2 타이밍 판정: "지금이 적기"

**너무 이르지 않은 이유**:
- LLM/RAG 기술이 상용 수준에 도달 (2023~2024 GPT-4, Claude 3 이후)
- MCP가 업계 표준으로 자리잡음 (Anthropic, OpenAI, Google, Microsoft 모두 지원)
- 2차 베이비부머 은퇴가 이미 시작됨 (2025~)
- "Eternal AI"가 2026년 트렌드 키워드로 부상

**너무 늦지 않은 이유**:
- 한국 시장에 직접 경쟁사가 없음
- Eternos는 "추모"에서 "생산성"으로 피봇 → 아직 "전문 지식 보존" 영역은 공백
- 2차 베이비부머 954만명의 은퇴는 향후 11년간 지속 → 시장 성장 장기 지속

### 6.3 경고 신호

- **2~3년 내 선점하지 못하면** 대형 플랫폼이 유사 기능을 출시할 가능성 높음
- MCP 생태계가 성숙하면서 진입 장벽이 낮아지고 있음 → 속도가 핵심

## 7. 종합 평가

### SWOT 요약

| | 긍정 | 부정 |
|---|------|------|
| **내부** | **강점**: MCP 기반 기술 스택 확보, 명확한 차별점 (판단체계 이식), 양면 수익 모델 | **약점**: 아카이빙 과정 인력 집약적, 초기 전문가 확보 어려움, 브랜드 인지도 부재 |
| **외부** | **기회**: 초고령사회 진입, 954만명 2차 베이비부머 은퇴, 기술 단절 사회적 위기, MCP 표준화 | **위협**: 대형 AI 기업 진입, 규제 불확실성, 전문가 참여 저항감 |

### 최종 시장성 점수

| 평가 항목 | 점수 (10점) |
|-----------|-------------|
| 시장 규모 및 성장성 | 8/10 |
| 경쟁 우위 | 7/10 |
| 수익 모델 현실성 | 7/10 |
| 타이밍 | 9/10 |
| 진입 장벽/해자 | 7/10 |
| 한국 시장 적합성 | 9/10 |
| **종합** | **7.8/10** |

### 핵심 결론

1. **시장은 존재한다**: 한국의 초고령사회 진입 + 베이비부머 대량 은퇴 + 기술 단절 위기가 만드는 구조적 수요
2. **타이밍이 좋다**: 기술(LLM/RAG/MCP)과 수요(은퇴 물결)가 동시에 성숙한 교차점
3. **차별화가 가능하다**: "추모/감성"이 아닌 "산업적 지식 활용"이라는 독자적 포지셔닝
4. **속도가 생명이다**: 2~3년 내 전문가 네트워크와 데이터 해자를 선점해야 함
5. **아카이빙 자동화가 스케일링의 열쇠**: 초기 인력 집약적 모델에서 반자동화 모델로의 전환이 수익성 결정
